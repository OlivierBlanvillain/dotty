6 Notation:
7 
8 We use BNF notation. Terminal symbols start with at least two
9 consecutive upper case letters. Each terminal is represented as a
10 single byte tag. Non-terminals are mixed case. Prefixes of the form
11 lower case letter*_ are for explanation of semantic content only, they
12 can be dropped without changing the grammar.
13 
14 Micro-syntax:
15 
16   LongInt       = Digit* StopDigit        // big endian 2's complement, value fits in a Long w/o overflow
17   Int           = LongInt                 // big endian 2's complement, fits in an Int w/o overflow
18   Nat           = LongInt                 // non-negative value, fits in an Int without overflow
19   Digit         = 0 | ... | 127
20   StopDigit     = 128 | ... | 255         // value = digit - 128
21 
22 Macro-format:
23 
24   File          = Header majorVersion_Nat minorVersion_Nat UUID
25                   nameTable_Length Name* Section*
26   Header        = 0x5CA1AB1F
27   UUID          = Byte*16                // random UUID
28 
29   Section       = NameRef Length Bytes
30   Length        = Nat                    // length of rest of entry in bytes
31 
32   Name          = UTF8              Length UTF8-CodePoint*
33                   QUALIFIED         Length qualified_NameRef selector_NameRef
34                   EXPANDED          Length qualified_NameRef selector_NameRef
35                   EXPANDPREFIX      Length qualified_NameRef selector_NameRef
36 
37                   UNIQUE            Length separator_NameRef uniqid_Nat underlying_NameRef?
38                   DEFAULTGETTER     Length underlying_NameRef index_Nat
39                   VARIANT           Length underlying_NameRef variance_Nat      // 0: Contravariant, 1: Covariant
40 
41                   SUPERACCESSOR     Length underlying_NameRef
42                   PROTECTEDACCESSOR Length underlying_NameRef
43                   PROTECTEDSETTER   Length underlying_NameRef
44                   OBJECTCLASS       Length underlying_NameRef
45 
46                   SIGNED            Length original_NameRef resultSig_NameRef paramSig_NameRef*
47 
48   NameRef       = Nat                    // ordinal number of name in name table, starting from 1.
49 
50 Note: Unqualified names in the name table are strings. The context decides whether a name is
51 a type-name or a term-name. The same string can represent both.
52

207   Category 1 (tags 1-49)   :  tag
208   Category 2 (tags 50-79)  :  tag Nat

213 Standard Section: "Positions" Assoc*
214 
215   Assoc         = Header offset_Delta? offset_Delta?
216   Header        = addr_Delta +              // in one Nat: difference of address to last recorded node << 2 +
217                   hasStartDiff +            // one bit indicating whether there follows a start address delta << 1
218                   hasEndDiff                // one bit indicating whether there follows an end address delta
219                                             // Nodes which have the same positions as their parents are omitted.
220                                             // offset_Deltas give difference of start/end offset wrt to the
221                                             // same offset in the previously recorded node (or 0 for the first recorded node)
222   Delta         = Int                       // Difference between consecutive offsets,
