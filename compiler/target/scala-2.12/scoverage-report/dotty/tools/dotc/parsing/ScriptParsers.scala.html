11 /** <p>Performs the following context-free rewritings:</p>
12  *  <ol>
13  *    <li>
14  *      Places all pattern variables in Bind nodes. In a pattern, for
15  *      identifiers <code>x</code>:<pre>
16  *                 x  => x @ _
17  *               x:T  => x @ (_ : T)</pre>
18  *    </li>
19  *    <li>Removes pattern definitions (PatDef's) as follows:
20  *      If pattern is a simple (typed) identifier:<pre>
21  *        <b>val</b> x = e     ==>  <b>val</b> x = e
22  *        <b>val</b> x: T = e  ==>  <b>val</b> x: T = e</pre>
23  *
24  *      if there are no variables in pattern<pre>
25  *        <b>val</b> p = e  ==>  e match (case p => ())</pre>
26  *
27  *      if there is exactly one variable in pattern<pre>
28  *        <b>val</b> x_1 = e <b>match</b> (case p => (x_1))</pre>
29  *
30  *      if there is more than one variable in pattern<pre>
31  *        <b>val</b> p = e  ==>  <b>private synthetic val</b> t$ = e <b>match</b> (case p => (x_1, ..., x_N))
32  *                        <b>val</b> x_1 = t$._1
33  *                        ...
34  *                        <b>val</b> x_N = t$._N</pre>
35  *    </li>
36  *    <li>
37  *       Removes function types as follows:<pre>
38  *        (argtpes) => restpe   ==>   scala.Function_n[argtpes, restpe]</pre>
39  *    </li>
40  *    <li>
41  *      Wraps naked case definitions in a match as follows:<pre>
42  *        { cases }   ==>   (x => x.match {cases})<span style="font-family:normal;">, except when already argument to match</span></pre>
43  *    </li>
44  *  </ol>
45  */
